# Lang Master
Language packs made easy

Lang master aids you in renaming every block, item, and UI element in Minecraft through a clean interface

How to use:
1. Download the latest copy from the <a href="https://github.com/Vaporwave-13/Lang-Master/releases">Releases</a> page.
2. Launch the .exe file, either press 'import' if you want to import your own existing language file, or continue if you want to use the preloaded one.
3. Use the search bar to find text you wish to change
4. Click "Export as pack" once you are done. type in a name for the pack, choose a picture for the pack icon, choose a location for it to be saved to then drag it into the texture packs folder. press "Open RP Folder" to be instantly taken there.

# Nerd Stuff
If you want to run the code yourself, you need Python 3.10+ and the CustomTinker library.
